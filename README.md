# AdventureWorks Sales Analytics Dashboard

This repository contains an interactive Power BI dashboard built using the AdventureWorks sales dataset.

The project was created as a guided learning project to practice business intelligence reporting, dashboard design, and data visualization in Power BI.

## Project Overview

The dashboard focuses on sales performance analysis and helps explore key business metrics such as revenue, profit, orders, return rate, product performance, customer segmentation, and geographic sales distribution.

## Dashboard Pages

### Exec Dashboard

The executive overview page presents the main business KPIs, including revenue, profit, orders, return rate, revenue trends, and top-performing products.

### Map

The map page shows the geographic distribution of orders by country and continent.

### Product Detail

The product detail page provides product-level analysis, including revenue, profit, orders, targets, and price adjustment scenario analysis.

### Customer Detail

The customer detail page focuses on customer-level analysis, including revenue per customer, customer segmentation by income level and occupation, and order metrics.
